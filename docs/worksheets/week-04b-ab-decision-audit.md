## Part 7: Common Decision Traps

### Check which traps affect you:

| Trap | Description | You? | Antidote |
|------|-------------|------|----------|
| **Analysis Paralysis** | Endless research, never decide | ☐ | Set decision deadline |
| **Consensus Seeking** | Waiting for everyone to agree | ☐ | Consult, but decide alone |
| **Sunk Cost** | Sticking with bad decisions | ☐ | Ask: "Would I start this today?" |
| **Recency Bias** | Overweighting recent events | ☐ | Look at longer time horizons |
| **Overconfidence** | Underestimating risks | ☐ | Pre-mortem exercise |
