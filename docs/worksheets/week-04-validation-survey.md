## Part 2: The 5-Question Validation Survey

### Question 1: Pain Point Identification
**"What is your BIGGEST frustration when it comes to [problem area]?"**

| Response Options | Purpose |
|------------------|---------|
| ☐ Option A: [specific frustration] | Validate pain point A |
| ☐ Option B: [specific frustration] | Validate pain point B |
| ☐ Option C: [specific frustration] | Validate pain point C |
| ☐ Option D: [specific frustration] | Validate pain point D |
| ☐ Other (please specify): _______ | Discover unknown pains |

**Why this question:** Validates that the problem you're solving is actually felt by your target market.

---

### Question 2: Current Solution
**"How do you currently solve this problem? What do you spend on it?"**

| Response Options | Purpose |
|------------------|---------|
| ☐ I don't solve it (I just deal with it) | No current solution = opportunity |
| ☐ I use [Competitor A] - I spend $___ | Price anchor + competitive intel |
| ☐ I use [Competitor B] - I spend $___ | Price anchor + competitive intel |
| ☐ I use a DIY/free solution | Price sensitivity indicator |
| ☐ Other: _______ | Discover unknown alternatives |

**Why this question:** Establishes price anchors and identifies competitive landscape.

---

### Question 3: Interest Level
**"If a solution existed that [specific benefit], how interested would you be?"**

| Response Options | Score |
|------------------|-------|
| ☐ Not at all interested | 1 |
| ☐ Slightly interested | 2 |
| ☐ Moderately interested | 3 |
| ☐ Very interested | 4 |
| ☐ Extremely interested - I would buy immediately | 5 |

**Why this question:** Gauges demand intensity. Target: 60%+ scoring 4 or 5.

---

### Question 4: Willingness to Pay (The Key Question)
**"If this solution delivered [specific outcome], how much would you expect to pay?"**

| Response Options | Purpose |
|------------------|---------|
| ☐ $0-$X (Low range) | Price-sensitive segment |
| ☐ $X-$XX (Mid-low range) | Value-conscious segment |
| ☐ $XX-$XXX (Mid range) | Sweet spot indicator |
| ☐ $XXX-$XXXX (Mid-high range) | Premium segment |
| ☐ $XXXX+ (High range) | High-ticket potential |

**Alternative: Open-ended**
"What would you expect to pay for this solution? $_____"

**Why this question:** The most critical data point. Determines your pricing strategy.

---

### Question 5: Objection Discovery
**"What would make you NOT buy this solution?"**

| Response Options | Purpose |
|------------------|---------|
| ☐ Price is too high | Pricing objection |
| ☐ I don't believe it would work | Trust objection |
| ☐ I don't have time to implement | Complexity objection |
| ☐ I've tried similar things before and failed | Past experience objection |
| ☐ I need to see proof/results first | Evidence objection |
| ☐ Other: _______ | Unknown objections |

**Why this question:** Identifies objections to address in marketing/sales.
