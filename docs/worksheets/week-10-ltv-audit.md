## Part 1: Understanding LTV

### The LTV Formula (Subscription/Recurring)

```
LTV = Average Revenue Per User (ARPU) ÷ Churn Rate
```

### Example:
- ARPU = $50/month
- Churn Rate = 5%/month
- **LTV = $50 / 0.05 = $1,000**
