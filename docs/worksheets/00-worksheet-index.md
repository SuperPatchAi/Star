## PDF Conversion

To convert these markdown files to PDF:

### Option 1: VS Code Extension
1. Install "Markdown PDF" extension
2. Open the .md file
3. Press Cmd+Shift+P → "Markdown PDF: Export (pdf)"

### Option 2: Pandoc (Command Line)
```bash
# Single file
pandoc week-01a-financial-ladder-assessment.md -o week-01a-financial-ladder-assessment.pdf

# All files (batch)
for f in week-*.md; do pandoc "$f" -o "${f%.md}.pdf"; done
```

### Option 3: Online Converters
- [Markdown to PDF](https://www.markdowntopdf.com/)
- [Dillinger](https://dillinger.io/)
