# CREATIVE DIRECTION

## The Vibe

**"The Quiet Professional"**

Think: McKinsey meets Monocle magazine meets private equity. This is NOT:
- Tony Robbins / "Unleash Your Power" energy
- Flashy webinar bro with gradients and stock photos
- Generic Canva template feel

This IS:
- Boardroom sophisticated
- Understated confidence
- Data-forward, framework-driven
- Premium but accessible
- Timeless, not trendy

## Mood Board Keywords

- Executive boardroom
- Private club
- Financial Times
- Navy blazer + gold cufflinks
- Leather-bound journal
- Whiskey neat
- The Room Where It Happens

## Visual References

| Reference | What to Pull |
|-----------|--------------|
| **McKinsey reports** | Clean data visualization, structured layouts |
| **Monocle magazine** | Typography hierarchy, sophisticated simplicity |
| **Berkshire Hathaway annual letters** | Straightforward, substance over style |
| **Kinfolk magazine** | Whitespace, breathing room, editorial feel |
| **Private bank materials** | Premium but not gaudy, trustworthy |

# BRAND ELEMENTS

## Color Palette

### Primary

| Color | Hex | RGB | CMYK |
|-------|-----|-----|------|
| **Deep Navy** | `#0D1B2A` | 13, 27, 42 | 90, 75, 50, 65 |
| **Rich Gold** | `#C9A227` | 201, 162, 39 | 15, 30, 95, 5 |
| **Warm White** | `#FAF9F6` | 250, 249, 246 | 2, 1, 3, 0 |

### Secondary

| Color | Hex | Usage |
|-------|-----|-------|
| **Charcoal** | `#2C3E50` | Body text, borders |
| **Slate Gray** | `#64748B` | Captions, secondary text |
| **Light Gold** | `#F5E6B8` | Highlight backgrounds (10% opacity) |

### Usage Rules

- **Navy** = Authority, premium, depth
- **Gold** = Accents ONLY (10% of design max)
- **White** = Breathing room, text on dark backgrounds
- **Never** = Bright colors, gradients, neon

# NOTES FOR DESIGNER

1. **Whitespace is a feature, not a bug.** Don't fill every inch. Let it breathe.

2. **Gold is an accent.** Use sparingly. It should feel like a highlight pen, not a highlighter explosion.

3. **The client is a CEO.** This should feel like something you'd see in a private equity boardroom, not a marketing funnel.

4. **Fill-in areas are important.** The worksheets need clear, generous space for handwriting.

5. **Test the print.** Actually print a few pages to make sure it looks good on paper.

6. **Interactive is optional.** If time allows, add fillable form fields. But a static PDF is fine.
