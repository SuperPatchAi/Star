# The $100M Blueprint — PDF Design Specification
## Visual Design Guide & Layout System

---

# 1. DESIGN OVERVIEW

## Brand Positioning
**"The Quiet Professional"** — Sophisticated, confident, understated luxury. This isn't a flashy "guru" brand. It's a serious CEO sharing operational frameworks.

## Design Philosophy

| Principle | Application |
|-----------|-------------|
| **Premium, Not Flashy** | Dark, rich colors. No bright gradients or "webinar bro" aesthetics. |
| **Clean & Structured** | Heavy use of whitespace. Clear hierarchy. Easy to scan. |
| **Data-Forward** | Tables, frameworks, assessments front and center. |
| **Action-Oriented** | Fill-in worksheets, checklists, clear next steps. |
| **Timeless** | Avoid trendy design elements that date quickly. |

---

# 2. COLOR PALETTE

## Primary Colors

| Color | Hex | Usage |
|-------|-----|-------|
| **Deep Navy** | `#0D1B2A` | Primary background (dark pages), headers |
| **Rich Gold** | `#C9A227` | Accents, highlights, key callouts |
| **Warm White** | `#FAF9F6` | Light backgrounds, body text on dark |
| **Charcoal** | `#2C3E50` | Secondary text, borders |

## Supporting Colors

| Color | Hex | Usage |
|-------|-----|-------|
| **Slate Gray** | `#64748B` | Tertiary text, captions |
| **Light Gold Tint** | `#F5E6B8` | Background highlights (5% opacity) |
| **Success Green** | `#22C55E` | Checkmarks, positive indicators |
| **Alert Red** | `#EF4444` | Warning callouts, avoid behaviors |

## Color Ratios

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   60% — Deep Navy / Warm White (primary backgrounds)    │
│   30% — Charcoal / Slate Gray (text, structure)         │
│   10% — Rich Gold (accents, CTAs, highlights)           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

# 9. TOOL RECOMMENDATIONS

## Design Tools (Choose One)

| Tool | Skill Level | Best For |
|------|-------------|----------|
| **Adobe InDesign** | Expert | Professional print quality, full control |
| **Canva Pro** | Beginner | Quick, template-based, team collaboration |
| **Figma** | Intermediate | Design system, export to PDF |
| **Affinity Publisher** | Intermediate | One-time purchase, InDesign alternative |
| **Google Slides** | Beginner | Simple, shareable, exportable |

## Recommended Workflow

### Option A: Fast (Canva)
1. Start with Canva's "A4 Document" or "Letter" template
2. Set up brand colors and fonts
3. Create master pages for each template type
4. Build out page by page
5. Export as PDF (both print and web versions)
6. **Time estimate: 4-6 hours**

### Option B: Professional (InDesign)
1. Set up document with master pages
2. Create paragraph and character styles
3. Build template pages first
4. Flow in content from markdown
5. Create interactive elements (fillable fields)
6. Export with presets
7. **Time estimate: 8-12 hours**
